# bridgerton-twitter-analysis
## Project Overview
This project analyzes over 216,000 tweets about the Bridgerton 
TV series on Netflix. The goal was to understand how Twitter 
users felt about the show and which characters were most popular.

## Tools Used
- Python (Jupyter Notebook)
- Pandas (data manipulation)
- TextBlob (sentiment analysis)
- WordCloud (word cloud generation)
- Plotly (data visualization)
- Tableau Public (interactive dashboard)

## Key Findings
- 45% of tweets were Positive
- 41% of tweets were Neutral
- 13% of tweets were Negative
- Daphne was the most talked about character with 9,110 mentions
- Simon was second with 8,273 mentions

## Dataset
Sourced from Kaggle: [TV Viewership Analysis Dataset](https://www.kaggle.com/datasets/alistairs/the-mandalorian-tweet-dataset)
